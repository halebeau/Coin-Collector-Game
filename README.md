# **Coin-Collector-Game**
A simple game created with HTML, CSS, and Javascript that you can play in the browser.

### Languages
- HTML
- CSS
- Javascript
- phaser.js
### Gameplay
- Use <left and >right arrows on keyboard to move character. 
- Use spavcebar to jump
- Coins = +10
- Poinson = -20
- Star = +20
- Win = +100
